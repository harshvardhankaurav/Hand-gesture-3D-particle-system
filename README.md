# Hand Gesture 3D Particle System

A real-time interactive 3D particle system using Three.js.
Particles are controlled using hand gestures via webcam.

## Features
- Hand controlled movement
- Rotation with hand motion
- Shape morphing
- Color & size change with depth
- Works with basic webcams

## Tech
- Three.js
- JavaScript
- WebGL
- MediaPipe Hands

## Run
Open index.html in a browser and allow camera access.
